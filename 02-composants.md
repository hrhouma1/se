
### Composants principaux d’un ordinateur

| Composant                          | Où il se trouve ?                                             | Rôle ultra vulgarisé                                       | Exemple concret pour eux                                   |
| ---------------------------------- | ------------------------------------------------------------- | ---------------------------------------------------------- | ---------------------------------------------------------- |
| **Carte mère**                     | Grande “planche verte” à l’intérieur                          | Relie **toutes** les pièces entre elles (routes + prises). | Sans carte mère, impossible de connecter quoi que ce soit. |
| **Processeur (CPU)**               | Sur la carte mère, sous un gros ventilateur                   | “Cerveau” : exécute les programmes, fait les calculs.      | Ouvre un logiciel, lance un jeu → le CPU travaille.        |
| **Mémoire vive (RAM)**             | Barrettes fines branchées sur la carte mère                   | “Mémoire de travail” temporaire, très rapide.              | Onglets ouverts dans le navigateur, programmes en cours.   |
| **Disque dur / SSD**               | Boîtier (HDD) ou module plat (SSD)                            | Stocke les données **même quand le PC est éteint**.        | Windows, documents, photos, jeux, tout est là.             |
| **Carte graphique (GPU)**          | Sur un grand connecteur de la carte mère (ou intégrée au CPU) | Spécialiste de l’affichage, des images, des jeux.          | Jeux vidéo, montage vidéo, double écran, etc.              |
| **Alimentation (PSU)**             | Boîte avec un ventilateur et un câble secteur                 | Transforme le courant de la prise murale pour le PC.       | Sans elle, rien ne s’allume.                               |
| **Boîtier**                        | Autour de tout le reste                                       | Protège les composants, permet de fixer les pièces.        | Tour du PC, boîtier mini, moyen, grand format.             |
| **Ventilateurs / refroidissement** | Sur le CPU, dans le boîtier                                   | Évacuent la chaleur pour éviter la surchauffe.             | Ventilos qui tournent quand le PC force.                   |

### Périphériques d’entrée / sortie

| Type              | Composant                       | Rôle simplifié                                 | Exemple concret                        |
| ----------------- | ------------------------------- | ---------------------------------------------- | -------------------------------------- |
| **Entrée**        | Clavier                         | Permet de saisir du texte, des commandes.      | Taper un mot de passe.                 |
| **Entrée**        | Souris / pavé tactile           | Permet de déplacer le pointeur, cliquer.       | Cliquer sur un bouton dans Word.       |
| **Entrée**        | Micro                           | Enregistre la voix, le son.                    | Appel Zoom, dictée vocale.             |
| **Entrée**        | Caméra (webcam)                 | Capture des images / vidéo.                    | Cours en visioconférence.              |
| **Sortie**        | Écran / moniteur                | Affiche l’image générée par l’ordinateur.      | Bureau Windows, vidéos, jeux.          |
| **Sortie**        | Haut-parleurs / casque          | Restituent le son.                             | Musique, notifications, vidéos.        |
| **Entrée/Sortie** | Clé USB / disque externe        | Permet de **lire** et **écrire** des fichiers. | Sauvegarder un devoir, copier un film. |
| **Entrée/Sortie** | Carte réseau (Ethernet / Wi-Fi) | Permet la connexion à Internet / réseau local. | Naviguer sur le web, Teams, Moodle.    |
